Project Information

  • Goal: Real-time Computer Vision
  • Tech Stack:

Project Overview

In this project, the goal was to detect and count objects in real-time. Three distinct techniques were implemented across different scenarios to showcase versatility in computer vision:

  • HSV Color Separation: Detecting objects based on specific color ranges.
  • YOLOv5: Utilizing deep learning for robust object recognition.
  • Motion Detection: Tracking objects based on frame-to-frame movement differences.
