Details

I was downloading songs and realized how repetitive and time-consuming the process was. After just a few songs, I was already bored with the 5 to 6 steps required for each download. That's when I decided to put my automation skills to the test.

Having prior experience in this field, I set out to automate the entire process. I had a solution that saved me from the repetitive "donkey work" and streamlined the process significantly.

Automation has not only saved my time but also made the entire process much more enjoyable. If you're facing similar repetitive tasks, I highly recommend exploring the power of automation.

Before the program:
  1. Copy the link of the song.
  2. Paste it on a website for download.
  3. Search and convert it to MP3.
  4. Click on download.
  5. Remove the website's name from the file name.
After the program:
  1. Simply copy the links of the song.
  2. Done.
Here's how it works:
  1. Using the 'pyperclip' library, all copied links are automatically saved in a text file, freeing me to focus on copying links.
  2. The program then follows these steps:
  3. Takes the link from the file.
  4. Uses the 'webbrowser' library to search for the song on a website.
  5. Utilizes 'opencv' and 'pyautoGUI' to find and click buttons.
  6. Implements a simple loop to remove the website's name while downloading.
