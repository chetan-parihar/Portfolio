Project Details

The objective of this project was to create an automated bot capable of playing the "Piano Tiles" game and achieving high scores without manual intervention. By leveraging Computer Vision (OpenCV), the program analyzes the screen in real-time.

How it Works

The logic is straightforward: the program identifies the specific pixel coordinates of the four lanes. It constantly monitors these coordinates for the color black (which represents a tile). When a black pixel is detected, PyAutoGUI triggers a click event at that exact location.

Challenges

As the game progresses, the speed of the tiles increases significantly. We encountered a performance bottleneck where the frame processing rate (FPS) of the screen capture could not keep up with the game speed at higher levels, leading to missed clicks.
