Details

This project was developed with the objective of controlling the mouse pointer using hand gestures. By utilizing the fingertip movements, users can manipulate the mouse pointer's position and perform click actions. The system captures live video of the user's hand gestures and responds accordingly. OpenCV, a popular computer vision library, was employed for all input operations, enabling efficient video processing and analysis. To facilitate the movement of the mouse pointer, the PyAutoGUI library was utilized.

One of the key components of this project is the MediaPipe library, which provides a robust solution for hand tracking. MediaPipe offers pre-trained hand landmark models, created and made open source by Google. These models accurately detect and track 20 key points on the hand, enabling precise gesture recognition and control of the mouse pointer.

The project not only offers practical functionality but also adds an element of enjoyment by allowing users to interact with the mouse using their fingers without physically touching any device. This hands-free approach to controlling the mouse pointer adds convenience and can enhance the overall user experience.
